## Contribute a Translation

The lessons were originally written in Japanese and translated into English.
**Translations into other languages are very welcome** — every new language helps the aira community grow.

**How to contribute:**

1. Create a folder `docs/lessons_XX/` using the [ISO 639-1](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes) two-letter code for your language (e.g. `lessons_FR` for French)
2. Use `lessons_EN/` as your source — translate the surrounding text, leave code, filenames, and URLs unchanged
3. Open a Pull Request against the `main` branch

**Current translations:**

| Language | Folder | Status |
|----------|--------|--------|
| Japanese (original) | [`lessons_JP/`](lessons_JP/) | Complete |
| English | [`lessons_EN/`](lessons_EN/) | Complete |
| *Your language* | `lessons_XX/` | *You could be next!* |

→ See [CONTRIBUTING_TRANSLATIONS.md](CONTRIBUTING_TRANSLATIONS.md) for the full guide and PR instructions.
